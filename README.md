# Rose vom Wörthersee - 16km Regatta Landing Page

A modern, responsive landing page for the "Rose vom Wörthersee" rowing regatta - a 16km long-distance race from Velden to Klagenfurt on Lake Wörthersee, Austria.

## Overview

This is a single-page HTML website showcasing the annual rowing regatta with:
- Hero section with countdown timer
- Event highlights and information
- Statistics display
- Call-to-action sections for registration
- Responsive design for mobile and desktop

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Dark Mode Support**: Built-in dark mode using Tailwind CSS
- **Modern UI**: Uses Tailwind CSS framework with custom configuration
- **Material Icons**: Google Material Symbols for iconography
- **Custom Typography**: Lexend font family for clean, modern look

## Tech Stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Google Fonts - Lexend](https://fonts.google.com/specimen/Lexend)
- [Material Symbols](https://fonts.google.com/icons)

## Usage

Simply open `index.html` in any modern web browser to view the landing page.

## Key Sections

1. **Navigation Header** - Sticky navigation with brand logo and menu
2. **Hero Section** - Full-screen hero with background image and registration badge
3. **Countdown Timer** - Event countdown with days, hours, minutes, seconds
4. **Highlights Grid** - Three-column layout showcasing race route, trophy, and community
5. **Statistics Bar** - Key numbers about the event
6. **Call-to-Action** - Registration prompts with buttons
7. **Footer** - Links, social media, and copyright information

## Event Details

- **Distance**: 16 kilometers
- **Date**: October 12, 2024
- **Route**: Velden to Klagenfurt
- **Categories**: Individual sculls, double sculls, and eights across all age groups
